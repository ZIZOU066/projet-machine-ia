# Assets placeholders

Ce dossier contient des placeholders utiles pour la version statique.

- `hero-poster.svg` : poster fallback pour la section hero (si vidéo indisponible)

## Remplacement recommandé

1. Remplacer les miniatures vidéo (`i.ytimg.com/...`) par des visuels de la page produit/source officielle.
2. Ajouter 2-4 clips courts optimisés (H.264, <4-6 MB chacun) si hébergement local.
3. Garder un `poster` léger (<300 KB) pour le LCP mobile.
